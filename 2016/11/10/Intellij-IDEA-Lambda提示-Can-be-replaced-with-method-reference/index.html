问题

在IDEA中使用lambda表达式有时会有这样的提示：Can be replaced with method reference…

解决方法

出现这种提示一般可以将表达式改为用::关键字

方法引用 等价的lambda表达式
String::valueOf x -> String.valueOf(x)
Object::toString x -> x.toString()
x::toString () -> x.toString()
ArrayList::new () -> new ArrayList<>()
